# L293D-library-arduino

A library for the L293D motor driver.

# Functions:

# motorSpeed():
Description:
sets the speed of the motor.

Syntax:
motor.motorSpeed(speed)

Parametres:
speed:the speed of the motor between 0 to 255.

Returns:
nothing(void).


# forward():
Description:
turns the motor forward.

Syntax:
motor.forward()

Returns:
nothing(void).


# backward():
Description:
turns the motor backward.

Syntax:
motor.backward()

Returns:
nothing(void).


# stopMotor():
Description:
stops the motor.

Syntax:
motor.stopMotor()

Returns:
nothing(void).
